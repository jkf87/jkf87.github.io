## 1. 핵심 프롬프트 10선 (차시별)

### P-01 | 회의 일정 통보 이메일 (차시 1)

**원본 프롬프트:**
```
[배경]
• 주최: 정기 팀 미팅
• 일시: 12월 20일 (목) 14:00-15:00
• 장소: 제3회의실
• 참석자: 마케팅팀 전체
• 안건: 분기 마케팅 전략 검토

[명령]
위 내용으로 팀원들에게 보낼 회의 일정 통보 이메일을 작성해주세요. 정중하고 명확한 톤으로 부탁드립니다.
```

**테스트 케이스:**
| 항목 | 내용 |
|------|------|
| 샘플 입력 | 위 배경 그대로 |
| 예상 출력 | 제목·인사·일시/장소/안건·마무리 포함 6-8문장 이메일 |
| 품질 기준 | ① 누락 항목 0개 ② 존댓말 일관 ③ 복붙 즉시 전송 가능 |

**평가: PASS** — 구조가 깔끔하고 현역 모델에서도 안정적으로 동작.

### P-03 | 까다로운 상황 — 거절/사과/설득 이메일 (차시 1)

```

• 대안: 다음 달 협업 가능
```

| 품질 기준 | ① 관계 훼손 최소화 ② 대안 제시 ③ 비즈니스 이메일 형식 |

- "상황 1/2/3" 각각에 대해 역할·출력형식·금지사항을 명시해야 함
