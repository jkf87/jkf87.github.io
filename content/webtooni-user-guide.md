# 웹툰이(Webtooni) 사용법 완전 가이드

## 🚀 빠른 시작

### 1. 웹툰이 접속

[웹툰이 홈페이지](https://webtooni.com)에 접속합니다.

### 2. 새 프로젝트 생성

- **캐릭터 스타일:** 원하는 스타일을 선택합니다 (예: 드림한 분위, 액션 캐릭터)
- **프롬프트 입력:** 캐릭터의 개요를 자연어로 묘사합니다 (예: "화려한 학교 교복을 입고 웃는 소녀")

## 🎯 캐릭터 생성 프로세스

### 1단계: 4컷 만화 그리기

웹툰이가 4장의 일러스트를 생성합니다.

**팁:**
- 캐릭터의 주제가 명확할수록 더 좋은 결과물
- 여러 번형으로 시도해보기
- 프롬프트에 캐릭터 이름을 포함시키기

### 2단계: 투명 이미지 만들기

생성된 일러스트에서 투명 배경을 추출합니다.

**팁:**
- 복잡한 배경일수록 투명 추출 퀄리티가 좋아집니다
- 배경 요소와 분리가 잘 되도록 프롬프트를 작성하기

### 3단계: 텍스트 레이어링

텍스트를 캐릭터에 자연스럽게 배치합니다.

**팁:**
- 글자 크기, 폰트, 색상을 미리 지정하기
- 다국어 지원되는 프롬프트 작성하기
- 줄바꿈과 효과 고려하기

### 4단계: 스케치 (Sketch)

캐릭터의 초기 스케치를 생성합니다.

**팁:**
- 스케치는 정교할수록 더 매력적입니다
- 기본 스케치를 수정하거나 새로 추가하기
- 관련 아이템/소품을 스케치에 함께 포함하기

### 5단계: 디테일드 디렉션 (Detailed Directions)

구체적인 디테일을 지시하여 캐릭터를 더 완성합니다.

**팁:**
- 명확하고 구체적인 디테일드일수록 좋은 결과물
- 순서대로 디테일드 작성하기 (예: 얼굴 → 눈 → 코 → 입)
- 캐릭터의 특징을 강조하는 디테일드 사용하기

### 6단계: 일관성 유지 (Consistency)

동일한 캐릭터가 여러 장면에서 등장하도록 요청합니다.

**팁:**
- 캐릭터의 이름이나 별명으로 일관성 유지
- 의상이나 배경이 급격하게 바뀌지 않도록 신경 쓰기
- 여러 버전을 생성하여 나중에 최고 결과물 선택

### 7단계: 최종 캐릭터

최종적으로 완성된 캐릭터 이미지를 받습니다.

## 💡 고급 팁

### 반복 생성 최적화

- **버전 관리:** 같은 설정으로 여러 번을 시도해보고 최고 결과물 선택
- **시간 절약:** 자주 사용하는 옵션(표정, 포즈)을 미리 저장하기

### 트레블슈팅 (Troubleshooting)

**이미지가 제대로 생성되지 않아요:**
- 캐릭터 생성을 다시 시도하기
- 디테일드를 더 구체적으로 작성하기

**원하는 결과물이 나오지 않아요:**
- 프롬프트를 더 명확하게 작성하기
- 원하는 스타일을 더 구체적으로 설명하기
- 다른 스타일 옵션을 시도해보기

**텍스트가 깨지나요:**
- 영어 외의 언어 사용 시도하기
- 텍스트 입력란에 특수문자가 아닌지 확인하기
