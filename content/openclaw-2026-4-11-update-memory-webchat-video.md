## 서론

OpenClaw 2026.4.11은 "에이전트가 더 많이 기억하고, 더 자연스럽게 소통한다"는 방향의 업데이트입니다. ChatGPT에서 쌓아온 대화 기록을 OpenClaw 기억 시스템으로 가져올 수 있게 되었고, 웹챗에서는 미디어·음성·임베드가 구조화된 버블로 표시됩니다. 영상 생성 도구는 참고 오디오와 적응형 비율까지 지원해 콘텐츠 제작 파이프라인이 한 단계 깔끔해졌습니다.

운영자에게는 OAuth 스코프 수정·오디오 전사 버그 픽스·타임아웃 안정화 등 "고쳐야 할 것들이 고쳐진" 업데이트이기도 합니다.

---

## 핵심 변경점

### 1. 기억 시스템 — Dreaming & Memory Wiki

| 변화 | 체감 |
|---|---|
| ChatGPT 대화 내역 가져오기(ingestion) | 기존 ChatGPT 사용자가 OpenClaw로 넘어올 때 맥락 손실 최소화 |
| Imported Insights 탭 | 외부에서 들여온 인사이트를 별도로 탐색·검색 가능 |
| Memory Palace 일기 서브탭 | 날짜별 기록이 자동 정리되어 "이날 무엇을 했지?" 조회가 쉬워짐 |

**한 줄 요약:** 더 이상 빈 상태에서 시작하지 않아도 됩니다.

### 2. 웹챗 & 컨트롤 UI

- **미디어·답장·음성 지시어 → 구조화된 채팅 버블:** 어시스턴트가 보내는 이미지·음성·답장이 깔끔하게 렌더링됩니다.
- **리치 임베드 태그 + 외부 URL 게이팅:** 외부 링크를 안전하게 삽입하고, 허용 도메인 관리가 가능합니다.

**한 줄 요약:** 채팅창이 드디어 "채팅답게" 보이기 시작했습니다.

### 3. 영상 생성(video_generate) 고도화

- URL만으로 생성 에셋 전달, 별도 다운로드·재업로드 불필요
- `providerOptions` 지원, 세부 프로바이더 설정 노출
- 참고 오디오(reference audio) 입력, 음성 톤이나 BGM을 기준으로 영상 생성
- 역할 힌트(role hints) · 적응형 종횡비, 용도에 맞는 비율 자동 선택
- 이미지 입력 상한 증가, 더 많은 참고 이미지를 한 번에 전달

**한 줄 요약:** 프롬프트 하나로 영상 만들던 시절에서, 오디오·이미지·비율을 조립하는 파이프라인으로 진화했습니다.

### 4. 플러그인 생태계

- **활성화/설정 디스크립터** 도입, 설치 후 어떤 설정이 필요한지 한눈에 파악 가능
- **Feishu · Teams 채널 개선**으로 엔터프라이즈 환경 연동 안정성 향상

### 5. 버그 수정 & 안정화

| 항목 | 영향 |
|---|---|
| OpenAI/Codex OAuth `invalid_scope` 수정 | Codex 연동이 더 이상 인증 단계에서 튕기지 않음 |
| 오디오 전사 수정 (OpenAI/Groq/Mistral) | 음성 인식 빈 응답·오류 해소 |
| macOS Talk Mode, 첫 권한 부여 후 즉시 시작 | 권한 허용하고도 재시작이 필요하던 문제 제거 |
| ACP 하위 진행 메시지 누출 억제 | 불필요한 중간 로그가 채널에 뜨지 않음 |
| 타임아웃·페일오버·WhatsApp·Telegram 안정화 | 장시간 실행·네트워크 끊김 상황에서의 회복력 향상 |

## 왜 중요한가 — 실무 관점 해설

**기억의 이식이 가능해졌다는 것**은 단순히 "가져오기" 기능이 생겼다는 뜻이 아닙니다. 다른 AI 서비스에서 축적한 맥락을 OpenClaw가 곧바로 활용할 수 있게 되면서, **마이그레이션 비용이 급격히 낮아졌습니다.** 팀 단위 도입을 고민 중이라면 이것만으로도 전환 이유가 충분합니다.

**웹챗 리치 버블은 사용자 경험의 분기점**입니다. 지금까지는 에이전트가 보낸 미디어가 링크나 첨부파일로만 보였다면, 이제는 진짜 채팅 UI처럼 렌더링됩니다. 이 변화는 고객 대면 챗봇·내부 도구 모두에서 "아, 이거 AI가 아니라 진짜 앱이네"라는 인상을 줍니다.

**영상 생성 파이프라인 강화**는 콘텐츠 크리에이터에게 직접 와닿습니다. 참고 오디오를 넣고, 여러 이미지를 참조하고, 비율을 자동 맞추는 흐름이 하나의 API 호출로 끝납니다. 별도 편집 도구를 거치지 않아도 되는 구간이 늘어났습니다.

**안정화 수정 묶음**은 눈에 띄지 않지만 업무가 멈추지 않게 만드는 변화입니다. OAuth 스코프 버그 하나로 Codex를 쓸 수 없었던 상황, 음성 전사가 빈 응답을 반환하던 문제, macOS에서 권한을 줬는데도 재시작이 필요했던 불편함, 이런 것들이 하나씩 사라질수록 에이전트를 도구가 아닌 동료로 신뢰하게 됩니다.

## 마무리

OpenClaw 2026.4.11은 화려한 기능 추가보다 **기존 기능의 완성도**와 **에이전트의 자율성**에 무게를 둔 업데이트입니다. 기억이 스스로 자라고, 대화가 자연스럽게 흐르고, 만들어진 결과물이 바로 전달되는 경험, 그 방향으로 한 걸음 더 나아갔습니다.

업데이트가 적용된 상태라면 새로운 Memory Palace 탭과 웹챗 리치 버블을 바로 확인해 보세요. 아직이라면 `openclaw update` 한 번이면 됩니다.

궁금한 점이나 실사용 후기가 있다면 커뮤니티에 공유해 주세요. 피드백은 다음 릴리즈의 우선순위를 정하는 가장 직접적인 신호입니다.
