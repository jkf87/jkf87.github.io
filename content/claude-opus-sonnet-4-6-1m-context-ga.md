# Claude Opus 4.6 / Sonnet 4.6 — 1M 컨텍스트 윈도우 GA, 표준 가격으로 제공

Anthropic이 2026년 3월 13일, **Claude Opus 4.6과 Sonnet 4.6의 100만 토큰(1M) 컨텍스트 윈도우를 일반 공개(GA)** 했다. 베타 테스트를 거쳐 정식으로 전환된 이번 업데이트의 핵심은 단순하다: **롱컨텍스트 프리미엄 없이, 표준 가격으로 전체 1M 윈도우를 사용할 수 있다.**

---

## 핵심 변경 사항

### 1. 하나의 가격, 전체 컨텍스트 윈도우

기존에는 200K 토큰을 초과하는 요청에 추가 요금이 붙었다. 이제 그런 프리미엄이 완전히 사라졌다.

| 모델 | 입력 (per M tokens) | 출력 (per M tokens) |
|------|--------------------|--------------------|
| **Opus 4.6** | $5 | $25 |
| **Sonnet 4.6** | $3 | $15 |

90만 토큰 요청이든 9천 토큰 요청이든 **토큰당 단가가 동일**하다. 요금 곱셈기(multiplier)가 없다.

### 2. 미디어 처리 한도 6배 확대

요청당 최대 **600개의 이미지 또는 PDF 페이지**를 처리할 수 있다. 기존 100개에서 6배 증가한 수치다. 대규모 법률 문서 분석이나 수백 장의 연구 논문 이미지를 한 번에 넘길 수 있게 됐다.

### 3. 베타 헤더 불필요

200K 토큰 이상의 요청이 자동으로 처리된다. 별도의 beta 헤더를 추가할 필요가 없다. 이미 베타 헤더를 사용하고 있었다면 무시되므로, **코드 변경 없이 자연스럽게 전환**된다.

### 4. Claude Code 통합

**Claude Code Max, Team, Enterprise** 사용자는 Opus 4.6으로 자동으로 1M 컨텍스트를 사용한다. 추가 사용량 차지 없이, 기존 구독에 포함된다.즉, 대규모 코드베이스 분석 시 context compaction이 크게 줄어든다.

---

## 성능: MRCR v2에서 78.3%

1M 토큰 컨텍스트가 유의미하려면, 모델이 긴 문맥 속에서 올바른 세부사항을 정확히 회상하고 추론할 수 있어야 한다.

**Opus 4.6은 MRCR v2(Multi-Round Coreference Resolution) 벤치마크에서 78.3%**를 기록했다. 이는 해당 컨텍스트 길이에서 프론티어 모델 중 가장 높은 점수다.

각 세대의 Claude 모델이 출시될 때마다 롱컨텍스트 검색 정확도가 꾸준히 향상되고 있다.

## 가용성

1M 컨텍스트 윈도우는 오늘부터 다음 플랫폼에서 사용 가능하다:

- **Claude Platform** (네이티브)
- **Amazon Bedrock**
- **Google Cloud Vertex AI**
- **Microsoft Azure Foundry**

Claude Code Max/Team/Enterprise 사용자는 Opus 4.6에서 별도 설정 없이 자동 적용된다.
